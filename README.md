# Exploree ET

**Your Gateway to Opportunities in Ethiopia**

Exploree ET is a smart opportunity discovery platform built to help individuals and organizations in Ethiopia find relevant life-changing opportunities without endless searching.

## Features

- **Jobs** - Find career opportunities matched to your skills
- **Tenders** - Access government and private tenders
- **Events** - Discover trainings, workshops, and networking events
- **Opportunities** - Scholarships, grants, fellowships, and more

## Tech Stack

- **React** with TypeScript
- **Vite** for fast development
- **Tailwind CSS** for styling
- **shadcn/ui** for UI components
- **Framer Motion** for animations

## Getting Started

### Prerequisites

- Node.js (v18 or higher)
- npm or yarn

### Installation

```bash
# Clone the repository
git clone https://github.com/Exploree-Solutions/exploree-solutions-hub.git

# Navigate to the project
cd exploree-solutions-hub

# Install dependencies
npm install

# Start the development server
npm run dev
```

The app will be running at `http://localhost:8080`

### Build for Production

```bash
npm run build
```
